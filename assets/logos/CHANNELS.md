# Channel Logo Reference

Place channel logo files in this directory using the channel ID as the filename.

## File Format
- **Extension:** `.webp`
- **Naming:** `{channelId}.webp`
- **Recommended Size:** 120x120px (displays at 60x60px)
- **Background:** Transparent
